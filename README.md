# Portfolio Twan van der Lee
Titel: Portofolio Twan van der Lee
Auteur: Twan van der Lee
Datum: laatste update: 18-11-2025
Github: github.com/TwanvanderLee
Beschrijving: Dit project bevat een reproduceerbare Bookdown-portfolio voor het vak workflows.
De structuur is ontworpen volgens Guerilla Analytics-principes.


## Projectstructuur

├── analyse/              # Analyses en scripts  
├── bookdown/             # Bookdown en HTML-bestanden  
│   ├── *.Rmd             # Rmarkdowns voor de hoofdstukken van het portfolio  
│   ├── *.rds             # Opgeslagen objecten  
│   └── _book/            # HTML-output van het boek  
├── data/  
│   ├── ruw/              # Onbewerkte data  
│   └── voorbewerkt/      # Bewerkte data  
├── docs/                 # Documentfolder   
├── outputs/  
│   ├── figuren/          # Figurenoutput  
│   └── tabellen/         # Tabeloutput  
├── R/                    # Functies en hulpscripts  
├── Portfolio_Twan.Rproj  # RStudio project  
└── README.md             # Dit bestand
