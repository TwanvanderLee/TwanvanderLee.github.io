5.2.1 Portfolio Twan van der Lee

Titel: Portofolio Twan van der Lee Auteur: Twan van der Lee Datum: laatste update: 18-11-2025 Github: github.com/TwanvanderLee Beschrijving: Dit project bevat een reproduceerbare Bookdown-portfolio voor het vak workflows. De structuur is ontworpen volgens Guerilla Analytics-principes.
